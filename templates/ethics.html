Security, Privacy, and Ethics

Security

  • Passwords are hashed before storage.
  • Role-based access limits what each user can do.
  • Input validation reduces invalid or harmful entries.
  • User activity is logged for accountability.

Privacy

  • Students only see their own information.
  • Staff and admins access records only for school purposes.
  • Only relevant personal data is stored.
  • Confidential data should not be shared outside the system.

Ethics

  • Use the system fairly and responsibly.
  • Do not modify records without authority.
  • Do not misuse monitoring features.
  • Be transparent about why data is collected.
